## Cel
Celem tego repozytorium jest przygotowanie szkieletu aplpikacji backendowej zawierajacej wszystkie narzedzia niezbedne do developmentu, testowania i procesow CI/CD.

## Requirements

- Docker Engine >=20.10
- Docker Compose >=2.17
- Bash Shell
